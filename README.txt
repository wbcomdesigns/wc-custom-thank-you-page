== Description ==
 
This plugin allows the site administrators to customize their Thank You & My Account page with some logo, thank you text. You can also have some products listed which allows customers to go for a new purchase and also the customers can share when they have currently purchased.
 
== Installation ==
 
1. Upload the entire `wc-custom-thank-you-page` folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
